Walker (Locomotion)

Single-agent bipedal locomotion using PyBullet physics. Teach an agent to walk with configurable rewards, domain randomization, and curriculum learning.

  • Observation: 13-dim (pose + velocity)
  • Action: 3-dim (force x/y + torque z)
  • Physics: PyBullet
⚔

Organism Arena (Multi-Agent)

Two-agent competitive arena with configurable battle rules, organism growth, and self-play training. PettingZoo parallel API.

  • Observation: 8-dim per agent
  • Action: 3-dim (move x/y + attack)
  • Multi-agent: PettingZoo
